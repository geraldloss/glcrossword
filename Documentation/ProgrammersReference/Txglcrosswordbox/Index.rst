tx_glcrossword_box
------------------

.. code-block:: php

	$m_arrContent[Index]

Array with the content stored in a special index.

There are 2 types of content objects:

- tx_glcrossword_content_answerfield
- tx_glcrossword_content_question



From this class are three different types of classes derived.
