What does it do?
----------------

This plug-in provides the possibility to create own crosswords on your homepage. You are able to
edit the crossword in the front-end and you can get hints for single questions. In the end you can
resolve the whole crossword and it shows you all mistakes an correct answers.

You are able to design your own crossword with own questions and answers. There are almost no
limitations to size or layout.
